**How are graceful degradation and service workers related?**

Gradeful degradation is the idea that if a certain feature of the app isn't working, it shouldn't break the entire webpage (basically to have more redundancy), it relates to the service worker since the service worker saves cached data in case the network fails, then the user won't be greeted with nothing/network error and the cached data is fetched, adding more redudancy to the web app in case a certain feature fails (in this case the wifi/network)
